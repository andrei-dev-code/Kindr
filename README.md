
# 🌈 Kindr

**Kindr** este o platformă modernă de dating și socializare dedicată comunității LGBTQ+ din România.

Misiunea noastră este să oferim un spațiu digital sigur, incluziv și autentic, unde oamenii pot construi relații, prietenii și conexiuni reale fără judecată.

## 💡 Despre proiect

Kindr este construit ca un ecosistem complet de socializare:

- ❤️ Dating inteligent
- 🔥 Hot or Not (descoperire rapidă de profiluri)
- 🤝 Matchmaker bazat pe compatibilitate
- 📰 Feed social
- 💬 Mesagerie privată
- 📅 Evenimente comunitare
- ✍️ Bloguri și articole
- 🛡️ Sistem de raportare și siguranță

---

## 🎯 Obiectiv

Crearea unei comunități sigure și moderne pentru persoanele LGBTQ+ din România, într-un mediu digital:

- sigur
- respectuos
- incluziv
- modern
